# BloodLink MERN

A blood donation management system built with the MERN stack (MongoDB, Express, React, Node.js).

## Project Structure
- **backend/**: Express.js API server
- **frontend/**: React + Vite application

## Features
- Donor Registration & Dashboard
- Hospital Management
- Emergency Blood Request Broadcasting
- Real-time Notifications
